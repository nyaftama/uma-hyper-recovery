# ウマ娘 なんでも超回復メーカー (UMAMUSUME HYPER RECOVERY BANNER MAKER)

「ウマ娘 プリティーダービー」の温泉郷シナリオでおなじみの「超回復」演出を再現できるファンメイドの画像メーカーです。  
お好みの画像とキーワードを組み合わせて、オリジナルの超回復画像をブラウザ上で簡単に作成・共有できます。

- **Webアプリ**: [https://nyaftama.github.io/uma-hyper-recovery/](https://nyaftama.github.io/uma-hyper-recovery/)
- **作者**: [@nyaftama](https://nyaftama.github.io/)

---

## 主な機能

1. **多彩な画像入力対応**
   - ファイル選択 / ドラッグ＆ドロップによるアップロード
   - クリップボードからの直接貼り付け（Ctrl+V / Cmd+V）
   - iPhone等で撮影された HEIC / HEIF フォーマット画像の自動変換に対応

2. **キーワード & 演出カスタマイズ**
   - 「超回復 ○○パワー発動!」のキーワードを自由に入力可能（最大20文字）
   - ワンタップで設定できるプリセットチップ（温泉、睡眠、休日、焼肉、ラーメンなど）を搭載
   - フォント読み込み遅延時などの対応として「再描画」ボタンを設置

3. **アスペクト比切替 & キャンバス操作**
   - 7種類のアスペクト比（1:1, 2:3, 3:2, 3:4, 4:3, 9:16, 16:9）を選択可能
   - アスペクト比や文字数に応じたテキストの自動はみ出し防止機能
   - ドラッグ操作で「文字」および「画像」の位置を自由に移動（移動対象切替機能付き）
   - 画像の回転ボタン（「画像を左回転」「画像を右回転」）
   - パーティクル（キラキラエフェクト）のランダムシャッフル機能
   - 文字サイズ、文字の上下左右位置、画像の拡大率を調整できる詳細設定パネル

4. **保存 & シェア**
   - 高解像度Canvasでの「超回復」画像生成
   - 画像ダウンロード & クリップボードへのコピー
   - X（旧Twitter）へのワンタップ投稿リンク生成

---

## 使い方

1. **画像を準備する**
   - 画像ファイルをドラッグ＆ドロップするか、クリップボードから貼り付けます。

2. **キーワードを設定する**
   - 発動させたいパワーの名称（例: 「温泉」「焼肉」など）を入力するか、プリセットボタンを選択します。

3. **レイアウトを調整する**
   - 必要に応じてアスペクト比を選択し、プレビュー上でドラッグして文字や画像の位置を調整します。必要に応じて「詳細設定」から拡大率や文字サイズを調整できます。

4. **生成・保存**
   - "「超回復」画像を生成" ボタンをタップして、完成した画像をダウンロードするか、クリップボードにコピーしてX等でシェアします。
